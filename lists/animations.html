basic animations

animation-name: Specifies the name of the animation defined using @keyframes. It associates an element with the animation sequence.

animation-duration: Sets the time it takes for an animation to complete one cycle, measured in seconds or milliseconds.

animation-timing-function: Determines the pace of an animation by defining the acceleration curve. Options include "linear," "ease," "ease-in," "ease-out," and more.

animation-delay: Establishes a delay before the animation starts, in seconds or milliseconds.

animation-iteration-count: Sets how many times the animation should repeat. Use values like a number, "infinite," or "alternate" for back-and-forth motion.

animation-direction: Specifies whether an animation plays forwards, backwards, or alternates between the two.

animation-fill-mode: Determines how an element should be styled before and after an animation. Options include "forwards," "backwards," "both," and "none."

animation-play-state: Controls whether an animation is running or paused. Use "running" or "paused" to start or halt the animation. 

#boxID{
  animation-name: ANIM-NAME; 
  animation-duration: 10s; 
}
